# Customer Churn Analysis

## 📌 Problem
Customers are leaving the company (churn), which affects revenue.

## 🎯 Goal
Analyze customer data to understand why customers churn and predict churn.

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## 📊 Key Insights
- Month-to-month contracts have highest churn
- New customers churn more than old customers
- Customers without tech support churn more
